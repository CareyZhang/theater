## 重建環境:
1. 安裝xampp後，先開啟xampp control panel將Apache和MySQL start
2. 將檔案放到C:\xampp\htdocs\theater目錄下
3. 在網址列打入[This link](http://localhost/phpmyadmin)，點擊資料庫，新建資料庫名稱theater，編碼與排序選擇utf8_general_ci，點擊建立
4. 點擊使用者，帳號為admin，主機為localhost，密碼為0000，全域權限全選，新增使用者
5. 到資料庫theater後，點擊匯入選擇檔案theater.sql後，點執行
6. 匯入成功之後到[This link](http://localhost/theater/)即可看到網頁
